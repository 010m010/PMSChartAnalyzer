# PMSChartAnalyzer
 9key/PMS 譜面専用のチャート分析アプリケーションです。  
 pmsファイルを読み込んで秒間密度チャートを確認できるほか、難易度表とbeatorajaのsongdata.dbを連携することでローカルフォルダのpmsファイルを一括分析、グラフ表示、一覧表示することができます。  
 譜面制作時の参考にしたり、眺めて楽しんだりと活用してください。  

## 主な機能
- `.pms` / `.bms` を読み込んで解析（！！！.bms形式も読み込めますが、**9keyモード以外は対応していません**！！！）
- 積み上げ棒グラフによる秒間密度チャートの確認
- 平均密度、体感密度、終端密度差、高密度占有率、突風度数etc...　のあらゆる指標で譜面の特徴を分析
- 難易度表URLを読み込み、各指標の難易度別分布を箱ひげ図および難易度ごと統計表で表示
- 難易度表分析結果をcsv形式でエクスポート可能

## 注意点
- 現状、LR2のsong.dbを読み込むことはできません。
- 終端範囲の定義にクリアゲージの仕様が関係していますが、このクリアゲージの仕様はbeatorajaのものを使用しています。
- アプリケーションを使用すると、ホームディレクトリに履歴情報や譜面情報などのキャッシュが生成されます。お手数ですがアンインストール時はこれらのファイルを手動で削除してください。
- キャッシュ生成先：     
   `Windows: C:\Users\\<ユーザー名>\\.pms_chart_analyzer\`   
   `macOS: /Users/<ユーザー名>/.pms_chart_analyzer/`  
   `Linux: /home/<ユーザー名>/.pms_chart_analyzer/`  

 

## 必要環境
- Python 3.10 以上のインストールが必要です。

## ダウンロード方法
- Download Zipからソースコードを直接ローカルに保存し解凍してください。

## インストール/起動方法
- 必要バージョン以上のPythonがインストールされていることを確認してください。
